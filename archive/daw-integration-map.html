פיצ׳ר סטטוס כלי / פרוטוקול הערות
Transport + Mixer Control
Play, Stop, Faders, Pan
 ✓ קיים OSC UDP Port 8000 Logic CS Plugin Logic accepts OSC via Control Surface plug-in. Predefined OSC paths only (e.g. /mixer/volume/volume1). Must use Logic's CS plugin API.
Custom OSC Paths
Free-form parameter mapping
 ✗ חסר N/A Logic ONLY accepts predefined OSC controls. Cannot add custom paths. Must map via MIDI CC workaround → OSCulator or NI Reaktor as bridge.
MIDI Remote Input
Notes, CC over network
 ✓ קיים IAC Driver Network MIDI (Apple) Apple's built-in Network MIDI (rtpMIDI compatible) works natively. No extra software needed on Mac.
Logic Remote Protocol
iPad-style full control
 ◆ Workaround Bonjour Reverse Engineer Logic Remote uses Bonjour discovery + proprietary OSC. Can be reverse-engineered (monitor port 8000) to replicate full Logic Remote control.
שמע Live Output
Full mix streaming
 ⚡ חלקי BlackHole Multi-Output Device Create Aggregate/Multi-Output device in macOS Audio MIDI Setup: output to both physical card + BlackHole. Logic Mac-only = BlackHole only needed.

Full mix streaming
 ⚡ חלקי VB-Cable ASIO4ALL FL mainly Windows — VB-Cable is the go-to. Route Master output to VB-Cable Input. StudioSync reads VB-Cable Output. Free solution.

🔴 מה חסר — צריך לבנות

אלה הדברים שלא קיימים off-the-shelf ו-StudioSync חייב לבנות אותם:

🎧 Audio Streaming Engine

CRITICAL

אין שום כלי קיים שלוקח שמע מ-Virtual Audio Device ומשדר אותו ב-WebRTC לדפדפן עם latency נמוך. זה הלב של StudioSync.

→ Node.js + WebRTC (node-webrtc) + BlackHole/VB-Cable capture

🌉 Universal DAW Bridge

CRITICAL

גשר שמתרגם בין OSC (Ableton) ← → MIDI HUI (Pro Tools/FL) ← → Logic CS Protocol. שפה אחת לכל ה-DAWs.

→ Node.js bridge: osc.js + midi npm + custom HUI emulator

🖥 Screen Capture + Remote Control

CRITICAL

אין כלי שמספק screen share + mouse/keyboard injection ספציפית ל-DAW. צריך lightweight agent שרץ ברקע.

→ Tauri (Rust) + getDisplayMedia API + uinput/CGEvent injection

📡 P2P Signaling Server

HIGH

שרת קטן שמחבר בין שני המחשבים לצורך WebRTC handshake. לא מחזיק audio — רק מתאם חיבור.

→ Node.js + Socket.io + Fly.io/Railway deployment (free tier)

🎹 Web MIDI Gateway

HIGH

מאפשר לשותף להשתמש ב-MIDI controller דרך הדפדפן ולהעביר את הנגינה ל-DAW הראשית.

→ Web MIDI API + WebRTC DataChannel → RTP-MIDI / virtual port

📦 Virtual Audio Installer

MEDIUM

משתמשים לא יודעים להתקין BlackHole/VB-Cable. StudioSync צריך Auto-installer שמגדיר הכל אוטומטית.

→ Tauri app → silent install + Audio MIDI Setup config via AppleScript/PowerShell
